### 00 - Open the console in your application directory.

### 01 - Install tools
````dotnet
dotnet tool install -g dotnet-ef --version 3.0.0
````

### 02 - Add initial migration
````dotnet
dotnet ef migrations add InitialCreate
````

### 05 - Update database
````dotnet
dotnet ef database update
````
